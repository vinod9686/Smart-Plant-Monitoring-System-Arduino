# 🌱 SmartSense: AI-Ready Plant Monitoring System

## Description
SmartSense is an automated plant monitoring and irrigation system built using Arduino. 
It uses soil moisture, light intensity, temperature, and humidity sensors to intelligently control a water pump and lighting system.

## Features
- Automatic irrigation based on soil moisture
- Light control using LDR sensor
- Temperature & humidity monitoring
- Real-time serial output
- Easy to extend to IoT/AI

## Tech Stack
- Arduino UNO
- DHT11 Sensor
- Soil Moisture Sensor
- LDR Sensor
- Relay Module

## Future Scope
- Mobile app integration (Blynk)
- Voice control
- AI-based irrigation prediction
